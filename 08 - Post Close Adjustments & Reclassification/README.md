# SOP 08 – Post Close Adjustments & Reclassification

## Overview
This folder contains the Standard Operating Procedure (SOP) and
supporting tools governing adjustments identified after a reporting
period has been closed.

The objective is to ensure corrections, reclassifications, and estimate
updates are properly authorised, documented, and traceable.

## Contents

### SOP
• **SOP_08_Post_Close_Adjustments_and_Reclassification.docx**  
Defines responsibilities, approval thresholds, documentation standards,
and posting requirements.

### Operational Checklist
• **SOP_08_Post_Close_Adjustments_and_Reclassification_Checklist.xlsx**  
A control log used to track identification, review, approval, and
posting of post-close adjustments.

## Key Areas Covered
• Error identification  
• Reclassification logic  
• Estimate revisions  
• Materiality assessment  
• Restatement requirements  
• Audit adjustments  
• Approval governance  

## How This Is Used
The SOP provides the framework for managing changes after close, while
the checklist ensures transparency, accountability, and audit defence.

## Outputs
• Approved adjustments  
• Clear audit trail  
• Controlled amendments to financial statements  

## Standards
IAS 8 – Accounting Policies, Changes in Accounting Estimates and Errors
